# EV Charging Station Density Analysis — Tampa, FL

GIS + Python analysis of EV charging station density in Tampa, Florida using GeoPandas and QGIS.

## Overview
This project analyzes the spatial distribution of EV charging stations and identifies clustering patterns in Tampa using kernel density estimation.

## Tools Used
- Python
- Pandas
- GeoPandas
- QGIS
- Jupyter Notebook

## Workflow
1. Data collection (EV stations dataset)
2. Data cleaning using Python
3. Conversion to GeoJSON
4. Visualization in QGIS
5. Kernel density heatmap creation
6. Map layout and export

## Key Insight
EV charging infrastructure is highly concentrated in central Tampa, with lower density in suburban and peripheral areas.
